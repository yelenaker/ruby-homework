# Лабораторна робота 6 — Часткове застосування (Currying)

## Мета роботи
Навчитися реалізовувати часткове застосування (currying) для лямбд/процесів у Ruby та перевірити її роботу на 3-аргументних функціях.

## Реалізація
- Створено лямбду `sum3 = ->(a,b,c){ a+b+c }`.
- Реалізовано функцію `curry3(proc_or_lambda)`, яка дозволяє часткове застосування по одному або кільком аргументам одночасно.
- Підтримуються виклики:

```ruby
cur = curry3(sum3)
cur.call(1).call(2).call(3)     #=> 6
cur.call(1, 2).call(3)          #=> 6
cur.call(1).call(2, 3)          #=> 6
cur.call(1, 2, 3)               #=> 6
cur.call(1, 2, 3, 4)            #=> ArgumentError (забагато)


Також перевірено на іншій лямбді:

f = ->(a,b,c){ "#{a}-#{b}-#{c}" }
cF = curry3(f)
cF.call('A').call('B', 'C')     #=> "A-B-C"

Результати експериментів

Часткове застосування працює коректно для будь-якої комбінації аргументів.

Виклик без аргументів повертає callable, який чекає три аргументи.

Передача більше 3 аргументів викликає помилку ArgumentError.
